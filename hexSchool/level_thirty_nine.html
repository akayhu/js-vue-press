# 第三十九關：箭頭函式(arrow function)

情境：

技術主管看到小杰第 18 關的 Code 後，覺得非常吐血。

他說現在公司都用所謂的箭頭函式(arrow function)，所以希望小杰也將 18 關的程式碼的 forEach 改為箭頭函式寫法，但小杰從來沒寫過什麼箭頭寫法，一起來幫幫小杰尋找資源吧！

# 問題

  1. 如果你是初學者，請回傳第 18 關的 forEach ，改用箭頭函式的寫法來撰寫
