# 第十六關：forEach 與 HTML 字串累加

# 題目

題目先寫了陣列 + forEach 的方式來累加 HTML 字串，再來顯示介面出來。範例題目

請你依照這邏輯，試著改寫物件格式，呈現的格式如下程式碼。

<ul>
  <li>小杰的身高為 178 公分，體重是 70 公斤，BMI 數據為 20，狀態為正常</li>
  <li>小美的身高為 150 公分，體重是 200 公斤，BMI 數據為 38，狀態為超重</li>
</ul>
